# Voca — Voice OS Landing Page

This is the Vite + React implementation of the Voca landing page.

## Getting Started

To run the site locally, follow these steps:

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173/`).

## Project Structure
- `src/index.css`: Contains the extracted global CSS.
- `src/App.jsx`: The main landing page view constructed from React components.
- `src/components/`: Modularized React components for specific sections.
